Title: Appointment Scheduler 

Purpose: This application allows appointments to be scheduled by users for customers, and associate the appointments with contacts. All data is recorded to the appropriate database tables.

Author: Nick Shaffer

Contact Information: nicksha@live.com

Version: 1.0

Date: June 14th, 2022

IDE: IntelliJ IDEA 2021.1.3 (Community Edition)

JDK: 17.0.2

JavaFX: javafx-sdk-17.0.2

Instructions: Start the application and you will be greeted with a login screen. Login with valid credentials (stored within the Users table) and you will be lead to the home page. From here, you can Add, update, and delete customer and appointment records. Additionally, you can view the reports page to see 3 reports.

Additional Report Included: A breakdown of the customer base by country. This is displayed in a PieChart for easy viewing.

MySQL Connector Driver: mysql-connector-java-8.0.25
